# 📷 スクショメモ

スマホで撮った写真をアルバムに保存せず、メモとして管理できるPWAアプリです。

## ✨ 機能

- 📷 **カメラ撮影 / ファイル選択** — 画像をアプリ内だけに保存（アルバム不要）
- 📝 **テキストメモ** — 画像にメモ・タイトルを添付
- 🏷️ **タグ分類** — タグでメモを整理
- 🔍 **検索** — タイトル・メモ・タグで検索
- 📤 **共有** — Web Share APIでテキスト＋画像を共有
- 📱 **PWA対応** — ホーム画面に追加してアプリのように使える
- 🌐 **オフライン対応** — Service Workerによるキャッシュ

## 🚀 使い方

### オンラインで使う（GitHub Pagesで公開する場合）

1. このリポジトリをフォーク or クローン
2. GitHub Pages を有効化（Settings → Pages → Branch: main）
3. スマホで `https://<ユーザー名>.github.io/<リポジトリ名>/` にアクセス
4. 「ホーム画面に追加」でアプリとして使える

### ローカルで使う

```bash
# Python がある場合
python -m http.server 8080

# Node.js がある場合
npx serve .
```

ブラウザで `http://localhost:8080` を開く

## 📁 ファイル構成

```
sukusho-memo/
├── index.html      # メインアプリ（すべてのコードを含む）
├── manifest.json   # PWAマニフェスト
├── sw.js           # Service Worker（オフライン対応）
├── icons/          # アプリアイコン（192px, 512px）
└── README.md
```

## 💾 データについて

- すべてのデータは **ブラウザのlocalStorage** に保存されます
- サーバーへの送信は一切ありません
- ブラウザのデータを削除すると消えるのでご注意ください
